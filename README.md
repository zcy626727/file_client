# 网盘

## 功能

- 用户服务
    - 登录
    - 注册
    - 修改用户信息：名称、头像等
- 文件服务（核心服务）
    - 个人文件
        - 上传文件
        - 新建文件夹
        - 重命名
        - 移动
        - 批量移动
        - 删除
        - 批量删除
    - 回收站
        - 删除
        - 批量删除
        - 恢复
        - 批量恢复
- 分享服务
    - 主题分享
    - 合集分享
    - 链接分享
        - 创建分享链接
        - 访问分享文件
        - 更改分享状态：开启/关闭
        - 保存分享文件
- 团队服务
    - 团队空间
      - 文件上传：本地文件/网盘文件
    - 用户组
    - 空间文件

## 环境
